Encryption

Encryption is the process of converting plain text (readable data) into cipher text (unreadable data) so that unauthorized people cannot understand it. Symmetric encryption means the same key is used for both encryption and decryption. AES works on blocks of data (typically 128 bits per block). Key sizes can be 128, 192, or 256 bits, which determines the strength of encryption.

Steps of Encryption
Key Expansion: The original key is expanded into multiple round keys.
Initial Round: AddRoundKey: Each block of plaintext is combined with a portion of the key.
Main Rounds (10, 12, or 14 depending on key size):
SubBytes: Each byte is substituted using a substitution box (S-box).
ShiftRows: Rows of the block are shifted cyclically.
MixColumns: Columns of the block are mixed (mathematical transformation).
AddRoundKey: Round key is added again.
Final Round (no MixColumns): SubBytes → ShiftRows → AddRoundKey
Output: The final cipher text is produced.
