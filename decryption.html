Decryption

Decryption is the reverse process where cipher text is converted back to plain text using the same key.

Steps in AES Decryption
1.Input Ciphertext – Encrypted text comes as input.
2.Initial Round
AddRoundKey 3.Main Rounds (Nr – 1 times)
Inverse ShiftRows Inverse SubBytes (uses inverse S-box) AddRoundKey Inverse MixColumns
4.Final Round Inverse ShiftRows Inverse SubBytes AddRoundKey
Output Plaintext – The original readable text is recovered.
InvShiftRows → InvSubBytes → AddRoundKey (without InvMixColumns).
